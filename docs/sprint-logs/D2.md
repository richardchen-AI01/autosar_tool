## 🐛 解决的具体问题（按时间）

| # | 问题 | 解决 |
|---|---|---|
| 1 | `ModuleNotFoundError: jinja2` | `pip install jinja2 lxml` |
| 2 | `ImportError: ModuleContextBase from Common.Context` | Common.Context 加 `ModuleContextBase = Context` 别名 |
| 3 | `TemplateNotFound: Template_Base.jinja` | 搬入 V25.10 模板到 core/Common/templates/ + 加 search path |
| 4 | `UndefinedError: 'DefaultCustomer'` | env.globals 注入 V25.10 默认值（DefaultCustomer='iSoft' 等）|
| 5 | `'MemIf' object has no attribute 'checkFee'` | _Container 加 getSubContainer 方法 |
| 6 | `BP.Det_DetGeneral missing` | 用 xdis 从 V25.10 BswPathConstant.pyc 抽 6870 条 BP enum 全量化 |
| 7 | `generateCode() takes 1 positional but 2 given` | 加 *args 支持子类传 incFileList |
| 8 | 行尾不一致 (LF vs CRLF) | 输出转 CRLF 匹配 V25.10 Windows 风格 |
| 9 | validator 自带 broken pyz_source 撞名 core | `validator/Common` → `docs/v25_pyz_source_reference/validator_Common` |
| 10 | NvMEcucPartitionRef 无值 → None → 不可迭代 | BswBase getAttrValue 'Ref' 后缀启发式返 [] |
| 11 | ECUC-REFERENCE-VALUE 元素未解析 (NvM 120 ref 丢) | loader 加 `<REFERENCE-VALUES>/<ECUC-REFERENCE-VALUE>` 解析 |
