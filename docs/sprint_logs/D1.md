## ⚠️ 已知问题

1. **`pyz_source/` 反编代码有 SyntaxError**：uncompyle6 在某些控制流上产出 `return    return True` 这种，
   不能直接执行。已经复制到 `validator/Common/` 作为**参考阅读**用，但**不是**运行时
   依赖。运行时用的是我们手写的 `core/Common/`。

2. **BP enum `.value` 与 `.shortName` 双属性**：当前实现绕了一圈（用 `_BPMember` wrapper），
   不完美但够 D1 用。M2 会用 metaclass 重写让 `.value` 直接是字符串路径、`.shortName` 是叶子名，
   完全匹配 V25.10 契约。

3. **`def_elements` 字典空**：`getBswContainerByEnum` 总是返 `[]`，所以 `MemIf().MemIfGeneral` 
   总是 `None`。这是预期的——M2 会接 ARTOP 把 ARXML 加载到 `def_elements`。

4. **D1 没碰 Eclipse RCP / Tycho**：那部分 PLAN §5 安排在 D1-D2 实例 C 战场，今天（单实例）
   没做。D2 排期需要补这部分。
